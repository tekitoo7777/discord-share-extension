# Discord Share Chrome Extension

現在のページをタグ付きでDiscordに共有するChrome拡張機能です。

## 📋 機能

- 🔗 **ワンクリックでDiscordに共有** - 現在のページのURL、タイトル、メモをDiscordに送信
- 🏷️ **自動タグ生成** - ドメインやページ内容から自動的にタグを生成
- ✏️ **タグ編集機能** - 自動生成されたタグの編集、カスタムタグの追加
- 📝 **メモ機能** - ページと一緒にメモを送信
- 📊 **送信履歴** - 最近送信したページの履歴を保存（最大100件）
- 🎯 **コンテキストメニュー** - 右クリックメニューから選択テキストやリンクを共有
- 🔧 **カスタマイズ可能** - よく使うタグの保存、Webhook URLの管理

## 🚀 インストール方法

### 1. 拡張機能のダウンロード
このリポジトリまたはフォルダ（`discord-share`）をローカルにダウンロードまたはクローンします。

### 2. Chrome拡張機能として読み込み

1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` を入力
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `discord-share` フォルダを選択

## ⚙️ 設定方法

### Discord Webhook URLの取得と設定

1. **Discord Webhook URLを取得**
   - Discordでメッセージを送信したいチャンネルを右クリック
   - 「チャンネルの編集」を選択
   - 左メニューの「連携サービス」→「Webhook」を選択
   - 「新しいWebhook」または「Webhookを作成」をクリック
   - Webhookの名前を設定（例：Chrome Extension）
   - 「Webhook URLをコピー」をクリック

2. **拡張機能に設定**
   - Chrome拡張機能のアイコンをクリック
   - 「設定」ボタンをクリック
   - Webhook URLを貼り付け
   - 「接続テスト」で動作確認
   - 「保存」をクリック

## 📖 使い方

### 基本的な使い方

1. 共有したいページを開く
2. Chrome拡張機能のアイコンをクリック
3. 自動生成されたタグを確認・編集
4. 必要に応じてメモを入力
5. 「Discordに送信」をクリック

### タグの編集

- **タグの削除**: タグの「×」ボタンをクリック
- **タグの追加**: 入力欄にタグを入力してEnterキー
- **サジェスト**: 入力中に候補が表示されます

### コンテキストメニューから共有

- ページ上で右クリック → 「Discordに共有」
- テキストを選択して右クリック → 選択テキストも一緒に送信
- リンクを右クリック → リンクURLも一緒に送信
- 画像を右クリック → 画像も一緒に送信

## 🏷️ 自動生成されるタグ

### ドメインベース
- GitHub: `#github` `#code` `#開発`
- YouTube: `#youtube` `#video` `#動画`
- Twitter/X: `#twitter` `#x` `#sns`
- Qiita: `#qiita` `#tech` `#技術記事`
- その他多数のサイトに対応

### コンテンツベース
- プログラミング言語の検出（JavaScript、Python、React等）
- キーワードの検出（tutorial、guide、error等）
- GitHub: リポジトリの言語、issue/PR/release等を検出
- YouTube: チャンネル名、カテゴリーを検出

## 🔐 セキュリティ

- Webhook URLは Chrome Storage API で安全に保存
- HTTPS通信のみ使用
- 必要最小限の権限のみ要求

## 🛠️ トラブルシューティング

### 送信できない場合
1. Webhook URLが正しく設定されているか確認
2. 設定ページで「接続テスト」を実行
3. Discordのチャンネル設定でWebhookが有効か確認

### タグが自動生成されない場合
1. ページの読み込みが完了してから拡張機能を開く
2. 一部のサイトではセキュリティ制限により情報取得が制限される場合があります
